Diese Website kann einfach auf Vercel deployed werden:

   ```bash
   npm i -g vercel
   ```

   - Verbinde dein Repository mit Vercel
   - Vercel erkennt automatisch Next.js und deployed die Website

3. **Via Vercel Dashboard:**
   - Gehe zu [vercel.com](https://vercel.com)
   - Importiere dein Git Repository
   - Vercel konfiguriert alles automatisch

Die Website wird automatisch bei jedem Push zu deinem Repository neu deployed.
